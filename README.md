한/영 전환을 하지 않고 영문 상태로 친 타자를 고쳐 줍니다.
지원하는 자판 내에서 어떤 영문 자판으로 어떤 한글 타자를 치든 바로잡아 줍니다.

※ 'mfsjea'은 영문 쿼티 상태에서 세벌식으로 '한영'을 타자한 결과입니다.

본 프로젝트는 [이 프로젝트](https://github.com/Lee0701/mfsjea)의 JVM 판입니다.
